🎮 Quick Start Guide

⬅️ ➡️ Arrow Buttons - Navigate through wallpaper collection
❤️ Heart Button - Save current wallpaper to favorites
📱 Apps Button - Launch micro-apps panel (bottom-right)
⚙️ Settings Button - Access all customization options (top-right)
🔍 Search Bar - Type and press Enter to search with your default engine

🖼️ Wallpaper System

Dynamic SpaceX collection, favorites system, custom folder support, and online APIs (Pexels/Unsplash) with auto-download organization.

💰 Currency Tools

Real-time converter (bottom-left) plus advanced calculator via Apps panel. Support for 12+ currencies with customizable base and targets.

🔗 Quick Links

Hybrid system combining top visited sites with custom shortcuts. Automatic favicon detection and easy management (5-10 links).

🕐 Time & Date

Customizable widgets with multiple styles, timezone support, and layout options. Toggle visibility and formats from settings.

📱 Apps System

Centralized launcher for micro-apps including currency calculator, wallpaper browser, and persistent pomodoro timer.

🍅 Persistent Pomodoro

Cross-tab productivity timer that follows you to Gmail, YouTube, any website. Draggable widget with session tracking.

⚙️ Settings

Comprehensive customization with theme control, widget toggles, wallpaper sources, currency preferences, and persistent storage.
